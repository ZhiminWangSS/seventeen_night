# Seventeen Night - 酒单网页

一个简约高端的酒吧酒单网页，适合部署为GitHub Pages网站。

## 功能特点

- 简约高端的设计风格，符合酒吧氛围
- 响应式布局，适配各种设备
- 平滑滚动和过渡动画
- 分类展示酒品（鸡尾酒、威士忌、葡萄酒、精酿啤酒、无酒精饮品）
- 优雅的交互效果

## 技术栈

- HTML5
- CSS3 (使用CSS变量和Grid布局)
- 原生JavaScript
- Google Fonts
- Font Awesome图标

## 部署到GitHub Pages

1. 将此仓库重命名为 `yourusername.github.io`
2. 推送到GitHub
3. 在仓库设置中启用GitHub Pages
4. 选择主分支作为源
5. 访问 `https://yourusername.github.io` 查看网站

## 自定义内容

### 修改酒单内容

编辑 `index.html` 文件中的菜单项，更新酒品名称、价格和描述。

### 修改样式

编辑 `styles.css` 文件中的CSS变量来自定义颜色主题：

```css
:root {
    --primary-color: #1a1a1a;      /* 主色调 */
    --secondary-color: #d4af37;    /* 强调色 */
    --text-color: #f5f5f5;         /* 文本颜色 */
    --text-muted: #a0a0a0;        /* 次要文本颜色 */
    --bg-color: #0a0a0a;          /* 背景色 */
}
```

### 修改联系信息

编辑 `index.html` 文件中的footer部分，更新营业时间、地址和电话等信息。
